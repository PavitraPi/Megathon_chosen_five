# Customer Churn Prediction Dashboard

Interactive dashboard for explainable customer churn prediction analysis.

## Live Demo
Access the live dashboard at: https://churn-prediction-insights.streamlit.app

## Features
- Real-time churn prediction
- SHAP value explanations
- Feature importance visualization
- Individual customer analysis
- Risk assessment metrics

## Local Development
1. Clone the repository
```bash
git clone https://github.com/vyshh09/Megathon_chosen_five.git
cd Megathon_chosen_five
```

2. Install dependencies
```bash
pip install -r requirements-streamlit.txt
```

3. Run the app
```bash
streamlit run streamlit_dashboard_new.py
```

## Deployment
The app is automatically deployed via Streamlit Cloud when changes are pushed to the main branch.

## Configuration
Custom settings can be modified in `.streamlit/config.toml`
